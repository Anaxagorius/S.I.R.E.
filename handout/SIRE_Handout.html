What is S.I.R.E.?

S.I.R.E. is a web-based incident response training platform that lets instructors run live, scenario-driven emergency drills for groups of any size — no specialised hardware or software installation required.

Trainees join from any device using a session code, receive real-time escalating events, log their actions, and receive instant feedback — all synchronised through a persistent WebSocket connection.

How It Works
1
Instructor creates a session — picks a scenario and shares the 6-character session code.
2
Trainees join instantly — enter the code and display name in any browser; no account needed.
3
Scenario runs automatically — timed events escalate and broadcast to every participant in real time.
4
Trainees log actions — responses and rationale are captured and visible to the instructor live.
5
Instructor injects events — custom escalations or curve-balls can be broadcast at any moment.

Key Design Decisions
  • Socket.IO over plain WebSocket — built-in rooms, reconnection & namespacing provide seamless multi-session isolation.
  • In-memory session store — zero external dependencies; ideal for short-lived academic drills with no persistence overhead.
  • JSON scenario definitions — instructors and non-developers can author or modify scenarios without touching code.
  • Vite + React frontend — instant HMR in dev; optimised static bundle at build time, served from a global CDN.
  • API-key + ticket-ID auth — lightweight, auditable security model suited to academic and demo deployments.
